Problem 1¶

Methods¶

For Problem 1, the reynolds number for each of the two cases was calculated and used to estimate the Boundary Layer Velocity Distribution. This was done by obtaining the estimates of the non-dimensional streamwise velocity $(\frac{u}{u_e})$, for both the laminar and turbulent cases for an array of $\frac{y}{\delta}$ between 0 and 1. These non-dimensional values were then dimensionalized by using the given velocites and the laminar and turbulent Blasius solution approximations for boundary layer thickness of a flat plate.

Equations¶

Reynold's Number: $$Re = \frac{\rho V_\infty L}{\mu}$$

Non-dimensional velocity as a function of non-dimensional height: $$\frac{u}{u_e}\bigg|_{lam} \approx \bigg(\frac{2y}{\delta} -\frac{y^2}{\delta^2} \bigg);\qquad \frac{u}{u_e}\bigg|_{turb} \approx \bigg(\frac{y}{\delta} \bigg)^\frac{1}{7} $$

Laminar and turbulent Blasius solution approximations for boundary layer thickness of a flat plate: $$\delta(x)_{lam} = \frac{5.0}{\sqrt{Re_x}}; \qquad \delta(x)_{turb} = \frac{0.16}{Re_x^\frac{1}{7}} $$

Skin Friction Drag: $$C_{f,lam} = \frac{1.328}{\sqrt{Re_c}}; \qquad C_{f,turb} = \frac{0.074}{Re_c^\frac{1}{5}} $$

Problem 2¶

Method¶

For this problem, the provided code was utilized. There were some adjustments made to the ViscousCalcs, and VIvI functions in order to calculate the wall shear stress and drag coefficient due to friction.

Equations¶

Displacement Thickness $\delta^*$: $$ H = \frac{\delta^*}{\theta} = 2.605 $$ Viscous contribution from friction: $$ C_f = \frac{1}{c}\int_0^c c_f dx; \qquad c_f = \frac{\tau}{\frac{1}{2}\rho_\infty V_\infty^2} $$ Wall Shear Stress $\tau$: $$ \tau = 0.332\frac{\mu u_e}{x}\sqrt{Re_x}$$ Momentum Thickness $\theta$: $$\theta = \frac{0.664x}{\sqrt{Re_x}} $$

This then leads to: $$ \tau = 0.213808\frac{\mu u_e}{x}\frac{x}{\theta}$$

def Cp2V(Cp, Vinf):
    """Return velocity magnitude computed from pressure coefficient using
    Bernoulli's principle
    """
    return np.sqrt(1 - Cp) * Vinf

#MOMENTUM THICKNESS FROM BLAUSIUS FLAT PLATE/VON KARMAN INTEGRAL RELATION
def MomThickness(x, u, Vinf, mu=1.79E-5, rho=1.225):
    """Compute momentum thickness at a given x-location along chord from
    equation derived by applying Blausius solution for flat plat to
    von Karmen integral relation for boundary layer.
    x --> coordinates along chord
    u --> surface velocity distribution as f(x) (from pot. flow soln)
    """
    #Compute momentum thickness at each X location (via von Karman relation)
    n = len(x) #number of x-locations
    c = max(x) - min(x)
    mom_thk = np.zeros(n) #initialize momentum thickness vector
    #Loop through each x-location
    for i in range(n):
        #integral portion is of surface velocity from LE up to current X
        integral = np.trapz(u[0:i] ** 8.210, x[0:i] * c)
        #momentum thickness from von karman
        mom_thk[i] = np.sqrt( (0.440 * mu) / (rho * u[i] ** 9.210) * integral )
    return mom_thk

The FixTE functions makes the trailing edge solvable in a potential flow.

def FixTE(x, z, tol=1):
    """The displacement thickness due to the boundary layer at the
    trailing edge is so large that the TE becomes too blunt to solve correctly
    in a potential flow solver.  This function modifies an airfoil geometry
    so that the trailing edge is marginally sharper.
    x,z --> coordinates for ONE side of an airfoil (upper OR lower)
    tol --> surface slope tolerance (currently not used)
    """
    c = max(x) - min(x) #chord length

    #USE DATAFRAME FOR CONVENIENCE
    df = pd.DataFrame({'x' : x, 'z' : z})

    #FIND INDEX OF 90% CHORD LENGTH
    ind = df.loc[df.x >= c*0.9].index[0]

    #MAKE SURFACE FROM IND TO TE A STRAIGHT LINE (WEDGE)
    #number of points back to calc slope
    back = 5
    #Slope of airfoil surface at ind
    m = (df.loc[ind, 'z'] - df.loc[ind-back, 'z']) / (
         df.loc[ind, 'x'] - df.loc[ind-back, 'x'])
    #Fill rear surface points with straight line
    df.loc[ind:, 'z'] = (df.loc[ind:, 'x'] - df.loc[ind, 'x']) * m + df.loc[ind, 'z']

    return np.array(df['z'])

The ViscousCalcs function finds information needed for the viscous part of VIvI through the panel method solution. It modifies the original geometry by obtaining and using the momentum thickness, displacement thickness, and velocity.

In [31]:
def ViscousCalcs(df, Vinf, mu, rho):
    """For a SINGLE surface (upper or lower):
    From an inviscid panel method solution,
    compute momentum thickness/displacement thickness from Blausius flat plate/
    von Karman integral relation.
    Add disp. thickness to original geometry to produce "displaced" geometry.
    Compute wall shear stress from momentum thickness.
    (Viscous portion of VIvI)
    df --> dataframe containing parameters for SINGLE surface
    """
    #COMPUTE DISPLACEMENT THICKNESS
    #Compute surface velocity from surface pressure distribution
    df['U'] = Cp2V(df.Cp, Vinf)
    #Compute momentum thickness from Blausius/von Karman
    df['theta'] = MomThickness(df.x, df.U, Vinf, mu, rho)    
    #Compute displacement thickness
    H = 2.605 #Shape Parameter for Blausius distribution
    df['delta_star'] = H * df.theta

    #ADD DISP. THICKNESS TO GEOMETRY (POTENTIAL FLOW DIV. STREAMLINE)
    #add or subtract delta* depending on surface side
    surfactor = -1 if max(abs(df.z)) > max(df.z) else 1
    #add disp. thickness to airfoil
    df['zdisp'] = df['z'] + df['delta_star'] * surfactor
    #Sharpen TE for better panel solution
    df['zdisp'] = FixTE(np.array(df['x']), np.array(df['zdisp']))
    df['tau'] = 0.213808*mu*df['U']/df['theta']
    df['cf'] = df['tau']/(0.5*rho*Vinf**2)
    
    return df

Lastly, the VIvI function is called to perform a single iteration of the viscous/inviscid interaction. It runs an inviscid XFOIL simulation with the given geometry. It then uses the results to compute the displacement thickness along the geometry, and adds this to the given geometry in order to create the resulting surface.

def VIvI(foilfile, alfa, niter=0, Vinf=1, mu=1.79E-5, rho=1.225):
    """SINGLE ITERATION of Viscous/Inviscid Interaction:
    Run invicid panel method simulation with provided geometry,
    compute displacement thickness of BL from surface pressure distribution,
    return "displaced" geometry, which can be used for another VIvI iteration
    foilfile --> path to airfoil geometry file
    alfa --> Single AoA at which to perform VIvI
    niter --> number for current iteration of VIvI
    Vinf, mu, rho --> freestream conditions
    """

    #GET INVISCID PANEL METHOD SURFACE PRESSURE SOLUTION

    #name and settings for specific iterations
    if niter == 0:
        #0th iteration
        num = ''
        pane = False
    else:
        #non-0th iteration
        num = '_{}'.format(niter)
        pane = True
    #name of current solution depends on VIvI iteration number
    geomfile = 'Data/{}/{}{}.dat'.format(foilfile, foilfile, num)
    #Run inviscid panel method
    invis = pyxfoil.GetPolar(geomfile, False, [alfa], Re=0, SaveCP=True, pane=pane, quiet=True)
    #Read panel method pressure distribution
    pressfile = 'Data/{}{}/{}{}_surfCP_Re0.00e+00a{:1.1f}.dat'.format(foilfile, num, foilfile, num, alfa)
    
    xCp = pyxfoil.ReadXfoilSurfPress(pressfile)
    #check of XFOIL solution failed
    if xCp.isnull().values.any():
        sys.exit('XFOIL solution failed for VIvI iteration: {}'.format(niter))
    
    #SPLIT MSES DATA INTO UPPER/LOWER SURFACES
    #Read current airfoil coordinates
    dfmses = pyxfoil.ReadXfoilAirfoilGeom(geomfile)
    #Compute chord length
    c = max(dfmses.x) - min(dfmses.x)
    #interpolate sufaces to this x
    xnew = np.linspace(0, c, 495)
    #store info for each surface in dictionary
    dfs = {'up' : pd.DataFrame(), 'lo' : pd.DataFrame()}
    #both sides have same x
    dfs['up']['x'], dfs['lo']['x'] = xnew, xnew
    #split surfaces
    dfs['up']['z'], dfs['lo']['z'] = mses.MsesInterp(xnew, dfmses.x, dfmses.z)
    #split surface pressure
    dfs['up']['Cp'], dfs['lo']['Cp'] = mses.MsesInterp(xnew, xCp.x, xCp.Cp)
    #save original geometry for plotting later
    dfog = dfs.copy()
    
    #VIVI COMPUTATIONS FOR EACH SURFACE
    for side in ['up', 'lo']:
        #compute displacement thickness from pressure distribution, add to geom
        dfs[side] = ViscousCalcs(dfs[side], Vinf, mu, rho)
        
    #MERGE AND SAVE DISPLACED GEOMETRY
    #Put displaced geometry into MSES format
    dfdisp = pd.DataFrame()
    dfdisp['x'], dfdisp['z'] = mses.MsesMerge(dfs['lo']['x'], dfs['up']['x'], dfs['lo']['zdisp'], dfs['up']['zdisp'])
    
    #Save displaced geometry in XFOIL format
    geomfile = 'Data/{}/{}_{}.dat'.format(foilfile, foilfile, niter+1)
    pyxfoil.WriteXfoilFile(geomfile, dfdisp.x, dfdisp.z)
    
    #RETURN UPPER/LOWER SURFACE MOMENTUM THICKNESS AND SURFACE VELOCITY DISTS.
    return dfs['up']['theta'], dfs['up']['U'], dfs['up']['tau'], dfs['up']['cf'], dfs['lo']['theta'], dfs['lo']['U'], dfs['lo']['tau'], dfs['up']['cf']

The reynolds number for the problem was 6.84e4. This is well under the critical reynolds number which means that it is laminar flow.

The VIvI results for the $C_l$ is in between the values that XFOIL gets for inviscid and viscous flow for the original geometry. The VIvI $C_l$ is most comparable to the viscous XFOIL, but it is not exactly the same. For the $C_d$, the case is the same but differently. The $C_l$ for the VIvI is higher than the Viscous XFOIl and lower than the Inviscid XFOIL. This is then backwards for the $C_d$. This is due to the fact of inviscid flow does not take into account drag. The difference between the VIvI and the Viscous XFOIL could be because the VIvI did not fully converge to a final solution. It could also be because the VIvI function is based entirely on inviscid flow and the Blassius solutions of a flat plate. For the $C_d$, however, it is so different because our VIvI method only takes into account skin friction drag, and the Viscous XFOIL takes more than that into account. The error is then exacerbated by the fact that the panel method used is flawed, as well as the fact that XFOIL has a much higher fidelity method.

The limitations of the VIvI tool come from the fact that it is based on inviscid flow and the blassius solutions only. For this project, the trailing edge bluntness is also a problem. The inviscid panel method used cannot take into account drag forces. It is also flawed since the number of panels is limited and the boundary conditions are only applied at the center of each panel. The fact that one of the panels has to be removed to satisfy the kutta-condition only makes this problem worse. This all adds up to the tool breaking down after a few iterations before the results fully converge.

This method would then only be appropiate for thinner airfoils are low angles of attack. This is mainly due to the flat plate Blassius solution. In turn, this method would be innaproppiate for high angles of attack due to separation issues from the APG as well as for high Reynold's Numbers where the Blassius solutions become invalid.

A CFD solver that solves the RANS time averaged equation would be a better higher fidelity method. While not perfect, a CFD solver would be able to simulate the flow much better than our VIvI function.

Conclusion¶

From this project, we can see how the viscous effects can be added to our previous vortex panel method. In order to do this, a Viscous/Inviscid Interaction tool (VIvI) is used. It can be seen in the results above that the Boundary Layer makes the airfoil thicker and decambered as more iterations are ran. The drag coefficient has to be computed separately from the momentum thickness and drag due to friction equations. This drag coefficient is significantly lower than Viscous XFOIL due to the fact that it is only skin friction drag. The implementation of the VIvI tool makes too many assumptions to be realistic and to have an useful range of iterations, however, it provides a simple understanding of viscous effects.
